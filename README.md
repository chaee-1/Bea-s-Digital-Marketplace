🛒 Bea’s Digital Marketplace
Bea’s Digital Marketplace is a modern, responsive e-commerce storefront designed to provide a seamless shopping experience. Built with Angular, this project showcases clean UI design, reactive data handling, and a scalable component-based architecture.

Key Features
Dynamic Product Catalog: Browse a variety of items with real-time filtering and category navigation.

Responsive Design: Fully optimized for mobile, tablet, and desktop viewing.

Interactive Shopping Cart: Add, remove, and manage items with instant price calculations.

Modern UI/UX: A clean, user-centric interface focused on conversion and ease of use.

Tech Stack
Frontend: Angular (v19+)

Language: TypeScript

Styling: CSS3 / SCSS

Testing: Vitest

Getting Started
To get a local copy up and running, follow these simple steps:

Clone the repo

Bash
git clone https://github.com/chaee-1/Bea-s-Digital-Marketplace.git
Install dependencies

Bash
npm install
Run the development server

Bash
ng serve
View in browser
Navigate to http://localhost:4200/.
